# Benny Wallet Downloads

Public Android APK downloads for Benny Wallet.

Release rules:
- Android APKs are published under versioned filenames, for example `releases/android/benny-wallet-v0.0.15.apk`.
- The `releases/android/` directory should contain only the latest APK and its matching `.sha256` file.
- `releases/app-update.json` must stay aligned with the current public app version.
